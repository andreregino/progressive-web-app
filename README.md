## PWA?

> Os Progressive Web Apps são um conjunto de técnicas para desenvolver aplicações web, adicionando progressivamente funcionalidades que antes só eram possíveis em apps nativos.

Em resumo, as principais características são:  

* Funciona offline
* Para qualquer usuário, em qualquer browser
* Responsivo
* App-like
* Atualização automática do app via browser
* Funciona somente com HTTPS
* Possibilidade de inclusão de push notifications
* Instalável no celular

### Inspiração

Repositório criado/baseado/inspirado no curso da [Alura de PWA ](https://www.alura.com.br/curso-online-progressive-webapp-offline).

### Instalação

Basta clonar o projeto, e executá-lo em um servidor Web qualquer, sem necessidades de instalar quaisquer dependências e/ou plugins, dado que o projeto usa Javascript nativo e as libs necessárias já estão no próprio projeto.
